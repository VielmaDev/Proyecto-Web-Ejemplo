Cómo clonar el proyecto.

1- Obtener la URL: Ir al repositorio remoto de GitHub y copiar la URL (HTTPS o SSH).
2- Abrir la terminal o Git Bash.
3- Navegar a la Carpeta usando cd para moverte a la carpeta donde quieres guardar el proyecto localmente (ej. cd C:\Proyectos\).
4- Para clonar se ejecuta el comando git clone [URL_COPIADA] y presiona Enter (ej. git clone https://github.com/usuario/nombre-del-repo.git)

Ejecutar el código en entorno local.

1- Entrar al directorio y navegar dentro de la carpeta que se acaba de crear con cd nombre-del-repo.
2- Instalar dependencia si aplica.
3- Iniciar la Aplicación con el comando de inicio (consultar la documentación del proyecto (README) o archivos de configuración (ej. python app.py)

Ramas del proyecto y su funcionalidad.

- main: rama principal del proyecto, la cual di inicio de la app y es la responsable de unificar todo el código para su exportación.
- feature-mejora-estilo: es una rama secundaria de apoyo para mejorar la apariencia del entorno, modificando principalmente el modulo index.css
